# QHay Web - Sistema de Gestión

Este es el frontend de la plataforma **QHay**, diseñado para la gestión de mesas, usuarios y restaurantes con una interfaz moderna y fluida.

## 🚀 Tecnologías Utilizadas

- **React 18**: Biblioteca principal para la interfaz.
- **Vite**: Herramienta de construcción ultra rápida.
- **Framer Motion**: Animaciones premium y transiciones suaves.
- **Lucide React**: Set de iconos elegantes y consistentes.
- **CSS Vanilla (Glassmorphism)**: Diseño moderno basado en transparencias y desenfoques.

## 🛠️ Cómo Iniciar el Proyecto Localmente

Sigue estos pasos para levantar la web en tu entorno de desarrollo:

### 1. Clonar el repositorio (si no lo has hecho)
```bash
git clone <url-del-repositorio>
cd qhay-web
```

### 2. Instalar dependencias
Asegúrate de tener [Node.js](https://nodejs.org/) instalado. Luego ejecuta:
```bash
npm install
```

### 3. Levantar el servidor de desarrollo
Para iniciar la aplicación, ejecuta:
```bash
npm run dev
```

Una vez ejecutado, la aplicación estará disponible en:
👉 **[http://localhost:5173](http://localhost:5173)**

### 4. Requisitos de la API
Para que los formularios (Usuarios y Restaurantes) funcionen correctamente, asegúrate de tener el backend corriendo en:
- API Base: `http://localhost:8080`

## 📂 Estructura de Vistas

- **Mesas**: Visualización y creación de mesas del sistema.
- **Usuarios**: Formulario de registro para nuevos integrantes/administradores.
- **Restaurantes**: Configuración de perfiles de negocio (Nombre, Dirección, Propietario, etc.).
